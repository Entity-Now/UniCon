# Siemens S7 驱动 (Siemens S7 Driver)

## 概述 (Overview)
西门子 S7 驱动实现了基于以太网对西门子全系列 PLC（如 S7-200 Smart, S7-300, S7-1200, S7-1500 等）的直连通讯。底层基于高效的 `S7.Net` 构建。

## 使用方法 (Usage)
通过 `S7Driver` 实例传入连接字符串建立连接，并执行读写操作或后台订阅。

## 参数说明 (Parameters)

### 连接字符串参数
| 参数名 | 类型 | 说明 | 是否必填 | 默认值 |
|--------|------|------|----------|--------|
| CpuType | `string` | 支持的值包括：`S7200`, `S7300`, `S7400`, `S71200`, `S71500` | 是 | 无 |
| Ip | `string` | PLC 的 IP 地址 | 是 | 无 |
| Rack | `short` | PLC 的机架号 (Rack) | 否 | 0 |
| Slot | `short` | PLC 的插槽号 (Slot) | 否 | 1 |

### 读写方法参数
| 参数名 | 类型 | 说明 | 是否必填 |
|--------|------|------|----------|
| Address | `string` | S7 寄存器绝对地址，如 `DB1.DBD0`，`M0.0` | 是 |
| value | `T` | 写入的具体值（仅 `WriteAsync` 需要） | 是 |

## 返回值 (Returns)

| 方法 | 返回类型 | 说明 |
|------|----------|------|
| `ReadAsync` | `UniconResponse<T>` | 内含值 `Data.Value` 及质量戳 `Data.Quality` |
| `WriteAsync` | `UniconResponse<bool>` | `true` 成功，`false` 包含错误信息 |
| `SubscribeAsync` | `Task<string>` | 返回订阅分配的 Guid 标识 |

## 使用示例 (Examples)

**示例 1：读取 S7-1200 实数**
```csharp
var s7Driver = new S7Driver("Line1_PLC", logger, cacheProvider);
await s7Driver.ConnectAsync("CpuType=S71200;Ip=192.168.0.10;Rack=0;Slot=1");

var req = new UniconRequest { Address = "DB1.DBD0" }; // 双字浮点数
var res = await s7Driver.ReadAsync<float>(req);

if (res.Success)
{
    Console.WriteLine($"值: {res.Data.Value}, 质量: {res.Data.Quality}");
}
```

**示例 2：后台订阅（v2 异常变化模式）**
```csharp
await s7Driver.SubscribeAsync(new UniconSubscription
{
    Address = "DB1.DBD0",
    ScanRateMs = 500,
    Callback = async dv => {
        Console.WriteLine($"[通知] {dv.Value}");
        await Task.CompletedTask;
    }
});
```
