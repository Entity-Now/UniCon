UniCon

高性能、插件化的 .NET 10 工业物联网通讯集成框架

屏蔽物理链路与底层协议差异，提供统一读写契约与极速 Channel 扫描引擎，为工业 4.0 与物联网提供强力支撑。

快速开始 →更新记录GitHub →（在新窗口打开）
🛡️

统一读写契约

借鉴类 Web 契约，引入 UniconRequest/Response 抽象，统一屏蔽 PLC、时序库及云端协议的物理差异。

⚡

v2 极速扫描引擎

基于 System.Threading.Channels 的异步无锁队列与分发，极致降低高并发大规模轮询下的资源消耗。

🔄

智能自愈守卫

内置 Watchdog 监控和指数退避重连算法，确保在恶劣的工业现场环境下仍能实现通讯自愈。

🕒

工业级定时调度

深度集成 Quartz.NET，支持 CRON 表达式及设备状态联动的复杂定时任务链和批处理调度。

🌐

动态 Minimal API

提供开箱即用的 WebAPI，支持在运行时动态注册/配置驱动，直接暴露标准 RESTful 数据接口。

📊

质量与时序溯源

DataValue 带有 Quality 数据质量戳和设备端 Server Timestamp，全面支持时序数据库清洗审计要求。
