## [v2.4.0] - 2026-05-19

### Added
- **VuePress + theme-plume 独立文档分支**：成功创建了纯净无源码污染的 `docs` 分支，配置了 VuePress + theme-plume 双主题高级静态博客与文档系统。
- **高性能 Git Worktree 本地文档同步脚本**：开发了 `scripts/sync-docs.sh` 脚本，在不切换开发分支的前提下直接于本地隐藏 worktree 进行文档自动对比、覆盖复制与提交。
- **自动触发 Git post-commit Hook 钩子**：编写并集成了本地的 `.git/hooks/post-commit` 钩子，实现了在主分支提交代码时自动触发文档向 docs 分支转换与更新的无缝开发流。
- **CI/CD 云端自动同步流**：配置了 `.github/workflows/sync-docs.yml` GitHub Actions 流水线，保证主分支文档有变动时远程云端完美同步并自动发布。
- **文档自动化同步设计方案**：编写了完备的 `/docs/1. introduce/3.docs-sync-design.md` 系统设计文档。

### Key Changes
- **Git 环境变量混淆防范**：在 sync 脚本中精细化解除了 Git Hook 运行时默认设置的 `GIT_DIR`、`GIT_INDEX_FILE` 等局部环境变量，完美规避了 Git 嵌套锁定与 "Not a directory" 冲突，达到了生产级的稳健性。
- **集合化 (Collections) 路由映射**：在 `plume.config.ts` 中精细化配置了 UniCon 五大技术文档子目录的集合路由，完美实现了去除数字前缀的前台友好排序链接，页面视觉美观典雅。
