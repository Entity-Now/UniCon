## [v2.4.0] - 2026-05-19

### Added
- **VuePress + theme-plume Docs Site**: Added a dedicated `docs` branch for hosting a beautiful, modern VuePress documentation site using theme-plume.
- **Git post-commit Hook**: Added a local Git post-commit hook that automatically syncs markdown documents from the `main` branch's `docs/` folder to the `docs` branch using Git Worktree.
- **Docs Synchronization Script**: Created `scripts/sync-docs.sh` to allow manual and automated documentation synchronization without switching the developer's working directory.
- **GitHub Actions Integration**: Added a GitHub workflow to sync documents automatically on remote pushes.
