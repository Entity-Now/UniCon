# Modbus 驱动 (Modbus Driver)

## 概述 (Overview)
Modbus 驱动实现了标准的 Modbus TCP 通讯协议。能够访问 PLC、变频器、智能仪表等广泛设备的线圈 (Coil)、离散输入 (Discrete Input)、保持寄存器 (Holding Register) 及输入寄存器 (Input Register)。

## 使用方法 (Usage)
实例化 `ModbusDriver` 并传入 IP 和端口连接，使用格式为 `[类型]:[地址]` 的地址串读取或写入。

## 参数说明 (Parameters)

### 连接字符串参数
| 参数名 | 类型 | 说明 | 是否必填 | 默认值 |
|--------|------|------|----------|--------|
| ip | `string` | Modbus 从站 (Slave) 或服务器的 IP 地址 | 是 | 无 |
| port | `int` | Modbus 服务的 TCP 端口 | 否 | 502 |
| unitid / slaveid | `byte` | 从站单元标识符 | 否 | 1 |
| timeout | `int` | 通讯及连接超时时间 (毫秒) | 否 | 2000 |

### 地址格式 (Address)
格式必须为 `[类型]:[偏移地址]`：
- `coil` (线圈, 读写布尔)
- `discrete` (离散输入, 只读布尔)
- `holding` (保持寄存器, 读写数值)
- `input` (输入寄存器, 只读数值)

示例：`holding:40001` 或 `coil:10`。

## 返回值 (Returns)

| 方法 | 返回类型 | 说明 |
|------|----------|------|
| `ReadAsync` | `UniconResponse<T>` | 内含值 `Data.Value` 及质量戳 `Data.Quality` |
| `WriteAsync` | `UniconResponse<bool>` | `true` 成功，`false` 包含错误信息 |

## 使用示例 (Examples)

**示例 1：读取保持寄存器**
```csharp
var modbusDriver = new ModbusDriver("Modbus_Meter", logger, cacheProvider);
await modbusDriver.ConnectAsync("ip=192.168.0.50;port=502");

var req = new UniconRequest { Address = "holding:1" };
var res = await modbusDriver.ReadAsync<int>(req);

if (res.Success)
{
    Console.WriteLine($"寄存器值: {res.Data.Value}");
}
```

**示例 2：使用统一调度引擎后台轮询仪表**
```csharp
await modbusDriver.SubscribeAsync(new UniconSubscription
{
    Address = "input:100",
    ScanRateMs = 1000,
    ScanMode = UniconScanMode.ExceptionBased,
    Callback = async dv => {
        Console.WriteLine($"表计改变: {dv.Value}");
        await Task.CompletedTask;
    }
});
```
