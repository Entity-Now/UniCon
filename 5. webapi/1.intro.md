# Web 接口与集成 (Web API & Integration)

## 概述
提供与外部系统交互的 RESTful 接口层，使用 Minimal API 设计。该模块充当南向工业设备与北向 IT 系统之间的通用协议转换与配置桥梁。

## 1. 整体职责
- **驱动生命周期管理**: 允许在不重启服务的前提下，通过 RESTful 接口动态添加、监测、修改及删除各类工业协议（如 S7、Modbus、OPC UA）驱动实例。
- **通用数据交互**: 屏蔽底层复杂的协议读取与写入差异，通过反射与类型转换机制，支持用统一的 HTTP 格式对任意设备点位地址进行读取和写入操作。
- **轻量级订阅值缓存**: 在内存中代理维持南向驱动的数据订阅，并在缓存中保存最新时间戳与数据质量，供北向客户端高速轮询。

## 2. 包含的主要功能
- **驱动注册与注销**:
  - `POST /api/drivers`: 注册并拉起新驱动。
  - `DELETE /api/drivers/{id}`: 停止、注销并注销现有驱动。
- **设备数据读写**:
  - `GET /api/drivers/{id}/read`: 读操作。
  - `POST /api/drivers/{id}/write`: 写操作。
- **数据订阅监测**:
  - `POST /api/drivers/{id}/subscribe`: 订阅点位。
  - `GET /api/drivers/{id}/subscriptions`: 轮询最新缓存。
