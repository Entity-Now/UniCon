# 驱动注册中心 (Driver Registry)

## 概述 (Overview)
驱动注册中心 `IDriverRegistry` 负责维护 UniCon 框架中所有活跃的通信驱动实例。它通过解耦驱动实例的创建与消费，支持在 WebAPI 交互层、定时调度系统 (Job System) 等多组件间实现驱动的全局共享和统一管理。

## 使用方法 (Usage)
在 .NET 依赖注入 (DI) 容器中将其注册为单例模式 (`Singleton`)：
```csharp
builder.Services.AddSingleton<IDriverRegistry, DriverRegistry>();
```
然后通过构造函数注入即可在控制器或定时作业中获取指定的驱动实例。

## 参数说明 (Parameters)

### Register 方法参数
| 参数名 | 类型 | 说明 | 是否必填 | 默认值 |
|--------|------|------|----------|--------|
| driver | `IUniconDriver` | 需要注册的驱动实例，必须继承自 `DriverBase` 并实现 `IUniconDriver` | 是 | 无 |

### Get 方法参数
| 参数名 | 类型 | 说明 | 是否必填 | 默认值 |
|--------|------|------|----------|--------|
| driverId | `string` | 目标驱动的唯一标识 ID | 是 | 无 |

## 返回值 (Returns)

### Get 方法返回值
| 类型 | 说明 |
|------|------|
| `IUniconDriver?` | 匹配 `driverId` 的驱动实例；若注册中心不存在对应 ID 则返回 `null` |

### GetAll 方法返回值
| 类型 | 说明 |
|------|------|
| `IEnumerable<IUniconDriver>` | 当前注册中心中所有已注册驱动的只读集合 |

## 使用示例 (Examples)

**示例 1：注册与获取驱动**
```csharp
// 注册驱动
var registry = new DriverRegistry();
var s7Driver = new S7Driver("PLC_Line1", logger);
registry.Register(s7Driver);

// 获取驱动
var retrievedDriver = registry.Get("PLC_Line1");
if (retrievedDriver != null)
{
    Console.WriteLine($"成功获取驱动: {retrievedDriver.DriverId}");
}
```
