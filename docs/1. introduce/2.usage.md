# 快速入门与集成指南

本指南将介绍如何在第三方项目中引用 UniCon 框架并快速实现工业设备通讯与任务调度。

## 1. 安装与引用

### 通过 NuGet (推荐)
在你的项目目录下运行以下命令（待包发布后）：
```bash
dotnet add package UniCon.Core
dotnet add package UniCon.Drivers.S7 # 仅引用需要的驱动
```

### 通过源码引用
1. 下载 UniCon 源码。
2. 将 `src/` 下的相关项目添加到你的解决方案。
3. 在你的主项目中添加对 `UniCon.Core` 的引用。

## 2. 基础通讯示例

以下展示如何初始化一个 S7 驱动并读取数据。

```csharp
using UniCon.Core;
using UniCon.Core.Models;
using UniCon.Drivers.S7;
using Microsoft.Extensions.Logging;

// 1. 初始化驱动
var logger = LoggerFactory.Create(builder => builder.AddConsole()).CreateLogger("S7");
var s7Driver = new S7Driver("PLC_01", logger);

// 2. 开启自愈重连配置 (默认开启)
s7Driver.EnableAutoReconnect = true;
s7Driver.InitialRetryIntervalMs = 2000;

// 3. 建立连接
await s7Driver.ConnectAsync("CpuType=S71200;Ip=192.168.0.10;Rack=0;Slot=1");

// 4. 结构化读取请求
var request = new UniconRequest { Address = "DB1.DBD0" };
var response = await s7Driver.ReadAsync<float>(request);

if (response.Success)
{
    Console.WriteLine($"Value: {response.Data.Value}, Timestamp: {response.Data.SourceTimestamp}");
}
else
{
    Console.WriteLine($"Error [{response.StatusCode}]: {response.ErrorMessage}");
}
```

## 3. 集成任务调度系统 (Quartz.NET)

UniCon 提供了开箱即用的 Job 系统。

### 注册服务
在 ASP.NET Core `Program.cs` 中：
```csharp
builder.Services.AddQuartz();
builder.Services.AddQuartzHostedService();
builder.Services.AddSingleton<IDriverRegistry, DriverRegistry>();
builder.Services.AddSingleton<JobScheduler>();
```

### 安排一个 HttpJob (云端联动)
```csharp
// 每分钟将设备状态发送至云端
await jobScheduler.ScheduleJobAsync<HttpJob>(
    jobId: "CloudSync",
    cronExpression: "0 * * * * ?",
    data: new JobDataMap 
    {
        [JobDataKeys.HttpUrl] = "https://api.cloud.com/sync",
        [JobDataKeys.HttpMethod] = "POST",
        [JobDataKeys.HttpBody] = "{\"status\": \"ok\"}"
    }
);
```

## 4. 批量操作与自愈监控

### 批量读取
```csharp
var requests = new List<UniconRequest>
{
    new() { Address = "DB1.DBD0" },
    new() { Address = "DB1.DBD4" }
};

var results = await s7Driver.ReadBatchAsync(requests);
foreach (var res in results)
{
    if (res.Success) Console.WriteLine(res.Data.Value);
}
```

### 状态监控
你可以订阅驱动的状态变更或定时检查 `State`：
```csharp
if (s7Driver.State == DriverState.Faulted)
{
    // 执行紧急逻辑，如切换备用通道
}
```
